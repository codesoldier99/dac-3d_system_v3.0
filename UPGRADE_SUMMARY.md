### 3. **数据库系统** 💾

**架构**:
- ✅ SQLite数据库（轻量、可靠）
- ✅ 5张核心表：
  - `batches`: 批次管理
  - `hole_results`: 孔位检测结果
  - `defects`: 瑕疵详细信息
  - `system_config`: 系统配置
  - `statistics`: 统计数据

**功能**:
- ✅ 批次管理（创建/完成/查询）
- ✅ 孔位结果存储
- ✅ 瑕疵详情记录
- ✅ 统计分析
- ✅ 历史数据查询

**代码位置**:
- `dac3d/services/database_service.py` (361行)

**使用示例**:
```python
db = DatabaseService("data/dac3d.db")
batch_id = db.create_batch("Batch_001", 144, "张三")
result_id = db.save_hole_result(batch_id, 0, 1, 1, True, 2, ["亮点", "暗点"])
db.save_defect(result_id, "亮点", (100, 200, 0), 15.5, 3500)
```

## 🎯 使用指南

### 快速开始

1. **安装依赖**
```bash
pip install -r requirements.txt
```

2. **运行系统（软件在环模式）**
```bash
python main_sil.py
```

3. **操作流程**
   - 点击"连接硬件" → 自动连接模拟设备
   - 点击"回零" → 系统初始化
   - 点击"开始检测" → 自动扫描144个孔位
   - 点击任意孔位 → 查看详情

```
Layer 4 (UI)

  ├── DefectService ← 新增

  ├── StateMachine
  └── EventBus

  ├── Real Drivers (ZMotion, PI, Basler)
  └── Sim Drivers ← 新增 (FPGA, Stage, Camera)

Layer 0 (FPGA)
  └── Verilog Code
```
