# 🚀 DAC-3D 快速启动指南

## 📋 系统说明

本系统支持两种运行模式：

1. **软件在环(SIL)模式** - 推荐用于开发和测试
2. **实际硬件模式** - 用于生产环境

## ⚡ 快速启动（软件在环模式）

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行系统

```bash
python main_sil.py
```

**注意**: 软件在环模式不需要连接真实硬件，所有设备都是模拟的！

```
└──────────────────┴──────────────────────────────────────┘
```

### 操作流程

1. **启动系统**: 运行 `python main_sil.py`

2. **连接硬件**: 点击"连接硬件"按钮
   - 软件在环模式会自动连接模拟设备

3. **回零**: 点击"回零"按钮
   - 系统初始化运动台位置

4. **开始检测**: 点击"开始检测"按钮
   - 自动创建新批次
   - 依次检测144个孔位
   - 实时显示检测进度
   - 瑕疵孔位显示为红色，合格孔位显示为绿色

5. **查看详情**: 点击任意孔位
   - 右侧显示孔位图像
   - 下方显示瑕疵详情

6. **导出报表**: 菜单 -> 文件 -> 导出报表

## 📊 颜色说明

| 颜色 | 状态 | 说明 |
|------|------|------|
| 灰色 | 未检测 | 尚未扫描该孔位 |
| 蓝色 | 检测中 | 正在扫描该孔位 |
| 绿色 | 合格 | 未发现瑕疵 |
| 红色 | 不合格 | 检测到瑕疵 |
| 橙色箭头 | 当前位置 | 实时扫描进度指示 |

## ⚙️ 参数配置

### 扫描参数配置

菜单 -> 设置 -> 扫描参数

- X步进: 扫描X方向步长(μm)
- Y步进: 扫描Y方向步长(μm)
- 扫描速度: 运动台速度(mm/s)

### 瑕疵检测参数

菜单 -> 设置 -> 瑕疵检测参数

- 检测阈值
- 最小瑕疵尺寸
- 最大瑕疵尺寸

## 🐛 问题排查

### 问题1: 导入错误

**症状**: `ModuleNotFoundError`

**解决**: 确保在项目根目录运行，并且已安装所有依赖

```bash
cd D:\research\福特科\核心算法\dac-3d_system_v3.0
pip install -r requirements.txt
```

### 问题2: 数据库错误

**症状**: 数据库文件锁定或损坏

**解决**: 删除数据库文件重新创建

```bash
rm data/dac3d_sil.db
```

### 问题3: UI显示异常

**症状**: 界面显示不正常

**解决**: 确保PyQt5正确安装

```bash
pip install --upgrade PyQt5
```
