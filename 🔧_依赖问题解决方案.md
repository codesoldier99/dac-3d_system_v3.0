# 🔧 依赖问题快速解决方案

## 问题描述

运行 `main_sil.py` 时出现错误：
```
ModuleNotFoundError: No module named 'transitions'
```

这是因为缺少Python依赖包。

---

## ✅ 解决方案（3种方式）

### 🚀 方式1：一键安装（最简单，推荐）

**直接双击运行**：
```
快速安装依赖.bat
```

这将自动安装软件在环模式所需的所有核心依赖。

---

### 🚀 方式2：使用精简版依赖（更快）

在命令行中运行：
```bash
pip install -r requirements_sil.txt
```

**优点**：
- 只安装必需的依赖
- 安装速度快（约2-3分钟）
- 适合软件在环模式

---

### 🚀 方式3：完整依赖安装（最全）

在命令行中运行：
```bash
pip install -r requirements.txt
```

**注意**：
- 包含所有可选依赖（napari、vtk、pynq等）
- 安装时间较长（10-15分钟）
- 适合完整开发环境

---

## ⚠️ 常见问题

### Q1: pip命令不存在

**解决**：
```bash
# 使用python -m pip代替
python -m pip install -r requirements_sil.txt
```

### Q2: 网络连接超时

**解决**：使用国内镜像源
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements_sil.txt
```

### Q3: 权限不足

**解决**：使用--user参数
```bash
pip install --user -r requirements_sil.txt
```

### Q4: Python版本过低

**解决**：
- 需要Python 3.9+
- 建议使用Python 3.10或3.11
- 下载地址：https://www.python.org/downloads/

## 📞 还有问题？

1. 查看详细日志，找到具体错误
2. 检查Python版本：`python --version`
3. 检查pip版本：`pip --version`
4. 尝试升级pip：`python -m pip install --upgrade pip`
