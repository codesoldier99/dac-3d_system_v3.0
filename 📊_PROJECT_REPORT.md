## 🎯 使用演示

### 完整操作流程

```
1. 启动
   ↓
   双击 START.bat
   
2. 连接
   ↓
   点击 [连接硬件]
   等待: "硬件连接成功"
   
3. 回零（可选）
   ↓
   点击 [回零]
   等待: "回零完成"
   
4. 开始检测
   ↓
   点击 [开始检测]
   
5. 观看魔术 ✨
   ↓
   • 144个孔位依次变色
   • 橙色箭头实时移动
   • 绿色=合格，红色=不合格
   • 进度条同步更新
   
6. 查看详情
   ↓
   点击任意孔位
   • 右侧显示图像
   • 下方显示瑕疵详情
   
7. 导出报表
   ↓
   菜单 → 文件 → 导出报表
```

**耗时**: 约10秒完成144孔检测（软件在环模式）
