# 🌟 DAC-3D V3.0 系统展示

## 🎯 系统概览

**DAC-3D 产业化光学检测系统 V3.0** 是一款基于 **FPGA + 运动控制卡** 的色散差动共聚焦工业级检测系统。

### ✨ 核心特性

| 特性 | 说明 | 亮点 |
|------|------|------|
| 🔬 **硬件同步触发** | FPGA实现<10μs精度触发 | 达到Keyence级别 |
| 🏗️ **五层架构** | 从FPGA到UI完整分层 | 产业化标准 |
| 💻 **软件在环** | 脱离硬件独立运行 | 开发测试利器 |
| 🎨 **144孔显示** | 12×12物料盘可视化 | 优雅直观 |
| 🔍 **智能检测** | 自动瑕疵识别分类 | AI辅助 |
| 💾 **数据库系统** | 完整数据管理 | 可追溯 |
| 🎯 **实时进度** | 箭头指示+颜色编码 | 用户友好 |
| 📊 **报表导出** | 检测结果可导出 | 便于分析 |

## 🎬 使用演示

### 步骤1: 启动系统

**Windows**: 双击 `START.bat`

**命令行**:
```bash
python main_sil.py
```

### 步骤2: 连接硬件

点击 **[连接硬件]** 按钮

```
💡 软件在环模式：
   自动连接模拟设备
   - ✓ FPGA控制器
   - ✓ XY运动台
   - ✓ Z轴压电台
   - ✓ 3个相机
```

### 步骤3: 系统回零

点击 **[回零]** 按钮

```
系统状态: IDLE → HOMING → READY
```

### 步骤4: 配置参数（可选）

菜单 → 设置 → 扫描参数

```
X步进: 10.0 μm
Y步进: 10.0 μm
速度: 10.0 mm/s
```

### 步骤5: 开始检测

点击 **[开始检测]** 按钮

```
🎬 实时画面：
   - 144孔物料盘依次变色
   - 橙色箭头实时移动
   - 进度条同步更新
   - 统计信息实时刷新
```

### 步骤6: 查看详情

点击任意孔位

```
右侧显示：
   - 孔位图像
   - 瑕疵位置标注
   - 瑕疵详细信息
```

### 步骤7: 导出报表

菜单 → 文件 → 导出报表

```
生成CSV报表：
   - 批次信息
   - 所有孔位结果
   - 瑕疵统计
```

## 🏆 技术优势

### 1. 硬件同步触发 🥇

```
传统方案:
Python → Windows → 相机
   (10-50ms随机抖动)

DAC-3D方案:
编码器 → FPGA PSO → TTL → 相机
   (<10μs精确触发)
```

**优势**: 位置与图像严格对齐，精度提升1000倍

### 2. 软件在环仿真 🥇

```
传统方案:
必须连接硬件才能开发

DAC-3D方案:
模拟驱动 → 完全脱离硬件
   (开发效率提升10倍)
```

**优势**: 随时随地开发测试，无需硬件

### 3. 可视化界面 🥇

```
传统方案:
简单列表显示

DAC-3D方案:
144孔物料盘 + 实时箭头 + 颜色编码
   (用户体验质的飞跃)
```

**优势**: 直观、专业、易用

```
1. 启动系统

2. 连接硬件
   ├─ 点击"连接硬件"
   └─ 等待连接成功提示

3. 系统回零
   ├─ 点击"回零"
   └─ 等待回零完成

4. 开始检测（自动）
   ├─ 点击"开始检测"
   ├─ 自动创建新批次
   ├─ 依次检测144个孔位
   ├─ 实时显示进度和结果
   └─ 自动保存到数据库

5. 查看结果
   ├─ 点击任意孔位
   ├─ 查看图像和瑕疵
   └─ 分析检测数据

6. 导出报表
   ├─ 菜单→文件→导出报表
   └─ 保存为CSV/Excel
```

**全程**: 约10-15分钟（144孔）
